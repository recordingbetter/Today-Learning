# 정규표현식 실습


### 실습 1  
{m}패턴지정자를 사용해서 a로 시작하는 4글자 단어를 전부 찾는다.

```python
re.findall(r'(?<=\s)[Aa]+\w{3}(?=\s)', story)
```
  
### 실습 2  
[]()r로 끝나는 모든 단어를 찾는다.

```python
re.findall(r'\w+?r\b', story)
```
  
### 실습 3  
a,b,c,d,e중 아무 문자나 3번 연속으로 들어간 단어를 찾는다.

```python
re.findall(r'\w*?[abcde]{3}\w*?(?=\s)', story)

```
